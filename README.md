# Countdown Timer Dock

This is a plugin for OBS Studio which adds a countdown timer dock that can be
used to automatically update a text source and trigger a scene change to a
target time of day.

It is also recommended to use the
[obs-text-directwrite](https://github.com/WamWooWam/obs-text-directwrite)
plugin on Windows, as you can use the OpenType Tabular Figures feature when
using a proportional font for your timer.

This plugin is based on the [obs-plugintemplate](https://github.com/obsproject/obs-plugintemplate). If you want to compile it yourself, please follow the build instructions there.
